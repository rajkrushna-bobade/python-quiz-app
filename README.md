# 🐍 Python Quiz App

An interactive Python programming quiz built with **HTML, CSS, and JavaScript**.

## Features

- 10 Python programming questions
- Multiple choice with 4 options each
- Instant feedback (correct/wrong highlight)
- Code snippets for relevant questions
- Explanation after each answer
- Score screen with performance feedback
- Fully responsive design

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Animations)
- Vanilla JavaScript

## Topics Covered

- Python syntax basics
- Data types (list, tuple, dict, string)
- Loops and range()
- Functions
- String methods
- Exception handling
- Operators

## How to Run

1. Clone the repository
```bash
git clone https://github.com/your-username/python-quiz-app.git
```
2. Open `index.html` in any browser
